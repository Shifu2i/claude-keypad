# Firmware

QMK firmware for the Claude Macropad.

**Status:** Coming in Sprint 2 — after PCB design is finalised.

## Planned Config

- Controller: Pro Micro (ATmega32U4)
- Matrix: 1×5 (5 switches, single row)
- Encoders: 2× EC11
- Left encoder: volume / scroll (default)
- Right encoder: configurable
- VIA support: planned
