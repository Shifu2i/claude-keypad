# Kickstarter Campaign Files

Campaign content for the Claude Macropad Kickstarter. All files are drafts — replace `[RENDER: ...]` and `[TBD]` placeholders before submitting.

## Files

| File | Purpose |
|---|---|
| `campaign-story.md` | Full Kickstarter page body — paste section by section into the Kickstarter editor |
| `reward-tiers.md` | Reward tier structure, pricing guidance, add-ons, and shipping table |
| `risks-and-challenges.md` | Mandatory Risks & Challenges field — paste directly into Kickstarter |
| `pre-launch-checklist.md` | Everything that must be done before the campaign can go live |

## Key Rules (Kickstarter Hardware)

- **You must have a working prototype before launching.** No exceptions.
- **No photorealistic renders as primary images.** Real prototype photos only.
- **Risks & Challenges is mandatory.** Be specific — vague text gets campaigns rejected or flagged by backers.
- **Fees are ~10% of funds raised** — budget for this in every unit price.
- **All-or-nothing:** you either hit your goal and collect everything, or collect nothing.
- **Estimated delivery dates are binding expectations.** Add a 2–3 month buffer to your best-case estimate.

## Placeholder Key

| Tag | Meaning |
|---|---|
| `[RENDER: ...]` | Replace with a real prototype photo or clearly-labelled render |
| `[TBD]` | Requires a real value before launch |
| `[Month Year]` | Fill in estimated delivery date (with buffer) |
| `[Creator name]` | Fill in real name |
