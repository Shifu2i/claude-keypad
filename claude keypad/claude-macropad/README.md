# Claude Macropad

> A fan-made mechanical macro pad for Claude Code power users.

![Claude Macropad Render](renders/render.png)

**5 keys · 2 rotary encoders · Claude orange · Robot-face silhouette**

---

## Overview

A themed mechanical macro pad designed for Claude Code workflows. Robot-face silhouette case in Claude orange (#DA7756), black PBT keycaps, and black anodised aluminium encoder knobs.

This is a fan project — not affiliated with or endorsed by Anthropic.

## Specs

| | |
|---|---|
| **Case** | FDM ASA, Claude orange (#DA7756) |
| **Keys** | 4× 1U + 1× 1.75U black PBT blanks (no stabilisers) |
| **Switches** | Hotswap — Kailh MX sockets |
| **Encoders** | 2× EC11 rotary with black anodised aluminium knobs |
| **Controller** | Pro Micro (ATmega32U4) |
| **Firmware** | QMK |
| **Connection** | Braided USB-C |
| **Dimensions** | 120 × 66 × 28 mm (desk to keycap top) |
| **Manufacturing** | JLCPCB (PCB + PCBA + case) |
