# Hardware

CAD and PCB design files for the Claude Macropad.

**Status:** In progress — Sprint 1.

## Case

- Material: FDM ASA
- Colour: Claude orange (#DA7756) — closest available JLCPCB filament
- Two-part construction: tray + top plate
- Wall thickness: 2.5 mm (3 mm at arm joints)
- Fasteners: M2 × 6 mm screws + M2 brass heat-set inserts

## PCB

- Tool: EasyEDA
- Controller footprint: Pro Micro (ATmega32U4)
- Switches: 5× Kailh MX hotswap sockets
- Encoders: 2× EC11
- PCB size: 114.5 × 49 mm (custom non-rectangular silhouette)
- Manufactured by: JLCPCB
